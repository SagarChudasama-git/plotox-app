## 🎯 About

Plotox is a powerful and intuitive tool designed to transform raw data from multiple formats (CSV, Excel, JSON) into beautiful, interactive charts and graphs. Whether you're analyzing business metrics, scientific data, or market trends, this platform makes data storytelling simple and impactful. Create, customize, and export professional visualizations in seconds.

---

## ✨ Features

### 📁 Multi-Format Data Support
- **CSV Import** - Seamlessly upload and parse CSV files
- **Excel Support** - Import XLS and XLSX spreadsheets with multiple sheets
- **JSON Format** - Direct support for JSON array and object structures
- **Drag & Drop** - Easy file upload with intuitive drag-and-drop interface

### 📈 Chart Types
- **Line Charts** - Track trends over time
- **Bar Charts** - Compare categorical data
- **Pie/Donut Charts** - Visualize proportions and percentages
- **Scatter Plots** - Analyze correlations and distributions
- **Area Charts** - Show cumulative trends
- **Histograms** - Display frequency distributions
- **Box Plots** - Analyze statistical distributions
- **Bubble Charts** - Multi-dimensional data visualization

### 🎨 Customization Options
- **Color Palettes** - Choose from predefined themes or create custom colors
- **Labels & Titles** - Add meaningful titles, axis labels, and legends
- **Grid & Styling** - Adjust grid lines, font sizes, and visual elements
- **Data Filtering** - Filter and sort data before visualization

### 💾 Export Capabilities
- **PNG Export** - Download charts as high-quality PNG images
- **SVG Export** - Scalable vector graphics for printing and web use
- **PDF Export** - Generate professional PDF reports
- **CSV Export** - Export processed data and calculations
- **Interactive Html** - Export Inteactive charts
- **JSON Export** - Save visualization configurations for later use

### ⚙️ Advanced Features
- **Real-time Preview** - See changes instantly as you modify data
- **Data Validation** - Automatic error detection and reporting
- **Statistical Calculations** - Built-in statistics and trend analysis
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Dark Mode** - Easy on the eyes for extended use
